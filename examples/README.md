# PixiJS skills — reference examples

Minimal demos that follow the skills: Application setup, sprites, graphics, text, ticker animation.

## Vanilla (HTML + JS)

- **examples/vanilla/** — single HTML page + ES module.
- Uses PixiJS from CDN (ESM). Open with a local server that supports ES modules (e.g. `npx serve examples/vanilla` from repo root).
- Patterns: Application init, Assets.load(), Sprite creation, Graphics with shape-then-fill, ticker-based animation.

These examples are intended as reference implementations for AI agents and for quick manual verification of the skill patterns.
